WORKFLOW

1 Plug device into any port (1-8)
2 Switch isolates device on dedicated VLAN
3 Pi detects link up, identifies device type
4 Pi uploads firmware + config automatically
5 Check web UI for status

TROUBLESHOOTING

No detection?Check cable, power, link LED on switch
Wrong firmware?Check firmware dir on Pi
Auth failed?Device has non-default password
Stuck?Restart provisioner-web service
Each port is ISOLATED - devices cannot see each other during provisioning
